# テニスのステップの種類と練習方法｜一人でできるフットワーク練習

テニスは「打つ」より「動く」ほうが先。ボールに合った場所に体を運べないと、どんなにフォームが綺麗でも安定しない。

この記事では、テニスでよく使う **5種類のステップ** とその **使い分け**、それから **一人でできる練習方法** をまとめておく。

---

## ① テニスでよく使うステップ5種類

![テニスのステップ5種類｜場面別の使い分け](/images/tennis-0428-step-types.svg)

ステップの基本は、この5つ。

1. **スプリットステップ**：構え直す
2. **サイドステップ**：近距離の横移動
3. **クロスステップ**：遠距離の高速移動
4. **ドロップステップ**：後退の1歩目
5. **リカバリーステップ**：中央へ戻る

順番に見ていく。

### 1. スプリットステップ（split step）

**いつ使う？** 相手が打つ瞬間。

両足で軽くジャンプして、相手のインパクトと **同じタイミング** で着地する。着地した瞬間に、左右どちらにも動ける状態になる。

- 高くジャンプしない（5cmくらい）
- 着地は **つま先** で、踵は浮かせる
- 着地した瞬間にもう次の動きへ

スプリットができていないと、反応が一瞬遅れる。「動き出しが遅い」と言われる人は、ほぼスプリットの問題。

### 2. サイドステップ（side step）

**いつ使う？** 1〜2歩分の横移動。

足を **交差させずに** 、横にスライドする。体は正面（ネット方向）を向いたまま。

- 視野が切れない（相手・ボールから目を離さない）
- 安定する（バランスが崩れにくい）
- ただし **遅い**。3歩以上はクロスステップに切り替え

ベースライン付近の細かい調整、ネットでのボレーポジションの修正、こういう場面に向く。

### 3. クロスステップ（cross step / crossover）

**いつ使う？** 3歩以上の横移動・遠いボール。

片足を反対側に **クロスさせて** 大きく動く。サイドステップより速い。

- 移動中、視野が一瞬切れる
- 速いけど、最後はサイドに戻して微調整
- 走り出しと止まり方の練習が必要

ワイドに振られた球、ベースラインからベースラインへの大移動はクロスで。

### 4. ドロップステップ（drop step）

**いつ使う？** 後ろ下がる時の1歩目。

片足を **後ろに引いて、体を斜めに開く** 動き。これだけで体の向きが変わって、後ろに走れるようになる。

- ロブが上がった時
- 深いボールを想定外に来た時
- 後退時、いきなり後ろ向きに走らない（バランス崩す）

ドロップステップ → クロスステップ → 打点で構え、という流れが多い。

### 5. リカバリーステップ（recovery step）

**いつ使う？** 打った後すぐ。

打ち終わった場所から、**コートの中央付近** に戻る動き。サイドステップやクロスステップを使う。

- 「打って終わり」をなくす
- ボールが返ってくるまでに **次のポジション** へ
- これがあるかないかで、試合の安定感が全然違う

上手い人は、必ずリカバリーしている。意識していない人は、必ず止まっている。

## ② 1ポイントでの順序

5つのステップは、1ポイントで **順番に使う**。

1. **スプリット**（相手が打つ瞬間）
2. **サイドまたはクロス**（球の場所へ移動）
3. **ドロップ**（後退する場合のみ）
4. **打つ**
5. **リカバリー**（中央へ戻る）
6. → また 1. のスプリットへ

このサイクルが回り続ける。1つでも抜けると、必ずどこかで崩れる。

## ③ 一人でできる練習方法4つ

![ステップ練習メニュー｜一人でできる4つのドリル](/images/tennis-0428-step-practice.svg)

ステップは **コートに行かなくても練習できる**。むしろ、コートで打つ前にやっておくべき基礎。

### 1. ラダードリル（5〜10分）

足元のラダー（縄ばしご）を使うトレーニング。Amazonで2,000円くらい。なければ、地面にチョークやテープでマス目を描いてもいい。

**メニュー例**

- 両足2歩（マスごとに左右1歩ずつ）
- 片足ケンケン
- サイドステップ（横向きで通過）
- イン-イン-アウト（マスに2歩、外に2歩）

ポイントは **接地時間を短く** すること。マスに足を置く時間を短くすればするほど、コートでの動き出しも速くなる。

### 2. シャドードリル（10〜15分）

ボールなしの素振りを **動きながら** やる練習。

**メニュー例**

- フォア5回 → スプリット → バック5回 → スプリット
- ベースラインの端から端までサイドステップ → 折り返し
- 前後の動き（前ボレー → 後ろリターン）

実際のラリーをイメージしながら動く。**スプリット → 移動 → 打つ → リカバリー** の流れを体に染み込ませる。

### 3. コーンドリル（10分）

コーンを5個くらい並べて、ジグザグに走る。コーンがなければ、ペットボトルでも空き缶でもOK。

**メニュー例**

- ジグザグダッシュ（コーンの間をクロスステップ）
- 各コーンで **方向転換** （サイドへの切り替え）
- 8の字走り（コーン2つの周りを8の字）

これは **方向転換の精度** を上げる練習。試合中、急に逆を突かれた時の対応力が変わる。

### 4. 階段ダッシュ（5分）

家の階段、駅の階段、神社の階段。なんでもいい。

**メニュー例**

- 1段ずつ全力で駆け上がる × 3〜5本
- 2段飛ばしで駆け上がる × 3本
- 下りはゆっくり歩いて休憩

これは **踵を踏む感覚** と **下半身の瞬発力** を鍛える。テニスのストロークで「左踵を踏む」感覚は、階段ダッシュで一番つかみやすい。

## ④ よくある質問（FAQ）

### Q. ステップの練習はどれくらいの頻度でやる？

週2〜3回でOK。毎日やる必要はない。1回30分以内、自分が続けられるペースで。

筋トレと一緒で、続かなければ意味がない。最初は週1でもいい。

### Q. ジムに行かないとできない？

不要。ラダーは家のリビングでもできる。階段はどこにでもある。コーンはペットボトルで代用できる。

道具より **習慣** のほうが大事。

### Q. ステップの練習で一番効くのはどれ？

人による。ただ、**初心者〜中級者なら ラダー → シャドー の順** が一番効く。

ラダーで足の細かい動きが速くなって、シャドーで実際のテニスの動きに変換する。この2つは、ほぼ全員に効く。

### Q. 試合でクロスステップを使ったほうがいい場面は？

ワイドに振られた時、ベースラインから3歩以上動かないと届かない時。サイドステップで間に合わない距離は、迷わずクロス。

ただし、**最後の1歩はサイドステップに戻す**。クロスのまま打つと、体が開きすぎる。

### Q. 後ろに下がる時に転びそうになる

それは **ドロップステップ** ができていない可能性が高い。後ろに走る時、最初の1歩は必ず **片足を引いて、体を斜めにする**。

いきなり後ろ向きに走ると、バランスが崩れる。ドロップ → クロス → 着地、の順を体に入れる。

### Q. リカバリーを忘れる

最初は誰でも忘れる。意識する方法は1つ。

**「打ったらすぐ動く」** を口に出す（または頭の中で言う）。打ち終わった瞬間に、もう次の場所へ動き出している、という意識を作る。

3回試合をやれば、習慣になる。

## まとめ

- テニスのステップは **5種類**（スプリット・サイド・クロス・ドロップ・リカバリー）
- 1ポイントで **順番に使う**（スプリット → 移動 → 打つ → リカバリー → スプリット）
- 練習は **一人でできる**（ラダー・シャドー・コーン・階段）
- 週2〜3回、自分に効くやつを継続

「打ち方」を練習する人は多いけど、「動き方」を練習する人は意外と少ない。だからこそ、ステップ練習をやれば差がつく。

ぼくも、ラダーを買って週2回やるようになってから、明らかに **試合中の動き出しが速く** なった。コートで打つ前の数分間、家でできる地味な練習。これが一番コスパいいと思う。
